Dein Veranstaltungsticket

Scanne den Code um dein Ticket einzulösen.

  • Öffne die Microsoft Authenticator App, die du bereits genutzt hast, um dein Veranstaltungsticket zu bekommen.
  • Klicke auf den Reiter „Überprüfe Ids“ in der Navigation (letzter Tab).
  • Klicke nun auf das kleine QR-Code-Icon unten Rechts in der App.
  • Scanne den QR Code.
  • Fertig.
